# 설악 잠실 버스 시간표


이 프로젝트는 가평 설악 터미널과 잠실역 간의 버스 시간표를 보기 어려우신 분들을 위해 시간표를 간단히 확인할 수 있는 웹사이트입니다.
웹사이트는 https://seorakBus.web.app 에서 접속할 수 있습니다.
